<i>Formerly Iconeration</i>. Icon pack creator for Android and Arcticons companion application. Create your own icon pack or extend Arcticons or another icon pack.

<b>Basic features</b>

<ul><li>Get icon from icon pack</li>
<li>Use path tracing on the application icon using Color Quantization</li>
<li>Use canny edge detection on the application icon</li>
<li>Create text from application name using Arcticons OTF font</li>
<li>Choose icon color</li></ul>

<b>Advanced features</b>

<ul><li>Use the vector version of the application icon if available</li>
<li>Use the monochrome version of the application icon if available</li>
<li>Export icons as themed icons</li>
<li>Import an image for a specific application</li>
<li>Create or edit the vector of the application icon</li></ul>
